Fibonacci vs Standard Grid Guide

Most CSS frameworks give you equal columns — 4+4+4 or 6+6. Vanduo goes further with Fibonacci-based layouts that follow the proportions found in nature: 1, 1, 2, 3, 5, 8, 13… These ratios create visual harmony that feels "right" even if you can't explain why. Let's explore the difference.

What is the Fibonacci Sequence?

Each number is the sum of the two before it:

1 · 1 · 2 · 3 · 5 · 8 · 13 · 21 · 34 · …

As the numbers grow, the ratio between consecutive numbers approaches ~1.618 — the Golden Ratio (φ). This ratio appears in sunflowers, seashells, galaxies, and the proportions of classical architecture.

Why does this matter for web design? Our brains are wired to find Golden Ratio proportions pleasing. Using Fibonacci ratios in your layouts creates a subtle sense of balance and sophistication that equal-width columns simply can't achieve.

Step 2 of 4 CSS Grid Fibonacci

Prefer CSS Grid over Flexbox? Vanduo has you covered with vd-grid-fib-* classes that use grid-template-columns with fractional units:

<!-- CSS Grid: 4-column Fibonacci (1fr 2fr 3fr 5fr) -->
<div class="vd-grid-fib-4">
    <div>1fr</div>
    <div>2fr</div>
    <div>3fr</div>
    <div>5fr</div>
</div>

<!-- CSS Grid: 3-column Fibonacci (3fr 5fr 8fr) -->
<div class="vd-grid-fib-3-alt">
    <div>3fr</div>
    <div>5fr</div>
    <div>8fr</div>
</div>
Flexbox vs CSS Grid? Both give you Fibonacci proportions. Use vd-row-fib-* (Flexbox) when you want wrapping behavior on small screens. Use vd-grid-fib-* (CSS Grid) when you want strict column placement and alignment across rows.

Step 4 of 4 The Grid Mode Toggle

Here's something unique to Vanduo: you can let users switch between Standard and Fibonacci grid at runtime with a toggle button. Great for dashboards or design tools where users want to control the layout:

<!-- Add this button anywhere in your page -->
<button data-grid-toggle aria-label="Toggle grid mode">
    <i class="ph ph-grid-four"></i> Toggle Grid Mode
</button>

<!-- Mark the section that should switch -->
<div data-layout-mode class="vd-container">
    <div class="vd-row">
        <div class="vd-col-4">Column 1</div>
        <div class="vd-col-4">Column 2</div>
        <div class="vd-col-4">Column 3</div>
    </div>
</div>

When the toggle is clicked, Vanduo swaps the grid from equal columns to Fibonacci proportions. The data-layout-mode attribute on the container tells Vanduo which section to transform.

Watch out: The grid toggle works on vd-row elements inside the data-layout-mode container. Make sure your columns add up to 12 in standard mode — Fibonacci mode recalculates proportions automatically.

When to Use Which

Use Standard Grid When…

  • Building data-heavy dashboards (equal emphasis on all metrics)
  • Creating forms with aligned columns
  • E-commerce product grids (uniform card sizes)
  • Any layout where symmetry is desired

Use Fibonacci Grid When…

  • Designing content-first pages (blogs, articles, portfolios)
  • Creating visual hierarchy (main content + sidebar)
  • Marketing / landing pages (guide the eye to the CTA)
  • Any layout where you want one area to dominate
Pro tip: You can mix both in the same page! Use a standard grid for your header and footer, and Fibonacci for the main content area. There's no rule against combining them.

Card Glow Effect

Add a gentle glow effect to any card with the .vd-card-glow class. On hover, cards emit a soft primary-colored glow using CSS box-shadow. 

<!-- Card with glow effect on hover -->
<div class="vd-card vd-card-glow">
  <div class="vd-card-body">Hover over this card</div>
</div>

Tip: The glow color adapts to your theme's primary color. Works beautifully with dark mode too!

<div class="vd-card">
    <div class="vd-card-header">
    <h3>Card Title</h3>
  </div>
    <div class="vd-card-body">
    <p>Card content goes here.</p>
  </div>
    <div class="vd-card-footer">
        <button class="vd-btn vd-btn-primary vd-btn-sm">Action</button>
  </div>
</div>

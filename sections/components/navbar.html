Navbar

Build responsive top navigation with brand, centered links, action buttons, mobile slide-in menus, and optional dropdowns.

Basic Structure

<nav class="vd-navbar">
  <div class="vd-navbar-container">
    <a href="#" class="vd-navbar-brand">Vanduo</a>

    <button class="vd-navbar-toggle" aria-expanded="false">
      <span></span><span></span><span></span>
    </button>

    <div class="vd-navbar-menu" aria-hidden="true">
      <ul class="vd-navbar-nav">
        <li><a class="vd-nav-link active" href="#">Home</a></li>
        <li><a class="vd-nav-link" href="#">Docs</a></li>
      </ul>
      <div class="vd-navbar-actions">
        <button class="vd-btn vd-btn-primary vd-btn-sm">Sign in</button>
      </div>
    </div>
  </div>
</nav>

Variants & Positioning

Class Purpose
.vd-navbar-darkDark theme navbar
.vd-navbar-transparentTransparent until .vd-navbar-scrolled
.vd-navbar-fixed / .vd-navbar-fixed-topFixed at page top
.vd-navbar-fixed-bottomFixed at page bottom
.vd-navbar-stickySticky while scrolling

Use .vd-navbar-dropdown, .vd-navbar-dropdown-menu, and .vd-navbar-dropdown-item for nested navigation menus.

Programmatic Control

Use the JavaScript API to add or remove draggable at runtime:

I'm not draggable yet... 
// Make any element draggable
const el = document.querySelector('#my-element');
VanduoDraggable.makeDraggable(el, {
  data: 'custom-data'
});

// Remove draggable behavior
VanduoDraggable.removeDraggable(el);

Accessibility

The draggable component is fully accessible with keyboard navigation and screen reader support.

Keyboard Shortcuts

Key Action
Tab Move focus between draggable items
Enter / Space Activate (click) the focused item
Arrow Up / Arrow Left Move item up / left within container
Arrow Down / Arrow Right Move item down / right within container
Escape Cancel current drag operation

ARIA Attributes (auto-applied)

  • role="option" on draggable items
  • role="listbox" on containers
  • aria-grabbed toggles during drag
  • aria-roledescription="draggable item" for screen readers
  • aria-dropeffect="move" on drop zones
  • tabindex="0" added automatically

Events Reference

All events bubble and include a detail object.

Event Fired On detail Payload
draggable:start Draggable element { element, data, position: { x, y } }
draggable:drag Draggable element { element, data, position, delta: { x, y } }
draggable:end Draggable element { element, data, position, delta }
draggable:drop Drop zone { zone, element, data, position }
draggable:reorder Draggable element { element, direction: 'up'|'down' } 
const el = document.querySelector('.vd-draggable');

el.addEventListener('draggable:start', (e) => {
  console.log('Drag started:', e.detail.data);
});

el.addEventListener('draggable:end', (e) => {
  console.log('Drag ended at:', e.detail.position);
});

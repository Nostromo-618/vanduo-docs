Base Markup

<div class="vd-code-snippet" data-collapsible>
  <button class="vd-code-snippet-toggle" aria-expanded="false">
    <span class="vd-code-snippet-toggle-icon"></span>
    <span>View Code</span>
  </button>

  <div class="vd-code-snippet-content">
    <div class="vd-code-snippet-header">
      <div class="vd-code-snippet-tabs" role="tablist">
        <button class="vd-code-snippet-tab is-active" data-lang="html">HTML</button>
        <button class="vd-code-snippet-tab" data-lang="css">CSS</button>
      </div>
      <button class="vd-code-snippet-copy" aria-label="Copy code">...</button>
    </div>
    <div class="vd-code-snippet-body">
      <pre class="vd-code-snippet-pane is-active" data-lang="html">...</pre>
      <pre class="vd-code-snippet-pane" data-lang="css">...</pre>
    </div>
  </div>
</div>

Options & API

  • data-collapsible: enables toggle behavior
  • data-expanded="true": expanded snippet state
  • data-visible="true": content visibility state
  • data-extract="#selector": extract HTML from page element
  • .has-line-numbers: add line numbers column
  • .vd-code-snippet-single, .vd-code-snippet-inline, .vd-code-snippet-compact: display variants 
// Programmatic control
CodeSnippet.expand(el);
CodeSnippet.collapse(el);
CodeSnippet.showLang(el, 'js');

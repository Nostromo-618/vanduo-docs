Search

A configurable client-side search component with keyboard navigation, debounced input, category grouping, and full accessibility support. Perfect for documentation sites, knowledge bases, or any content-heavy application.

Quick Start

Include the CSS and JS files, then add the search markup:

<!-- CSS -->
<link rel="stylesheet" href="css/components/doc-search.css">

<!-- HTML -->
<div class="vd-doc-search">
    <div class="vd-doc-search-input-wrapper">
        <i class="ph ph-magnifying-glass vd-doc-search-icon"></i>
        <input type="search" class="vd-doc-search-input"
               placeholder="Search..."
               aria-label="Search"
               aria-controls="search-results"
               aria-expanded="false"
               autocomplete="off">
        <kbd class="vd-doc-search-shortcut">⌘K</kbd>
    </div>
    <div class="vd-doc-search-results" id="search-results"
         role="listbox" aria-label="Search results"></div>
</div>

<!-- JS -->
<script src="js/components/doc-search.js"></script>

The component auto-initializes on page load and indexes all <section> elements with id attributes.

Features

  • Keyboard Navigation: Arrow keys, Enter to select, Escape to close
  • Keyboard Shortcut: ⌘K (Mac) / Ctrl+K (Windows) to focus
  • Debounced Input: 150ms delay prevents excessive searches
  • Category Grouping: Results grouped by section with icons
  • Highlight Matching: Search terms highlighted in results
  • ARIA Accessible: Full screen reader support
  • Dark Mode: Automatic theme adaptation
  • Reduced Motion: Respects prefers-reduced-motion

Custom Instance

Create a custom search instance with your own data and callbacks:

// Create custom search instance
const mySearch = Search.create({
    // Custom selectors
    inputSelector: '.my-search-input',
    resultsSelector: '.my-search-results',
    
    // Custom data source (optional)
    data: [
        {
            title: 'Getting Started',
            content: 'Learn how to install...',
            category: 'Guides',
            url: '#getting-started'
        },
        // ... more items
    ],
    
    // Callbacks
    onSelect: (item) => {
        console.log('Selected:', item);
        window.location.hash = item.url;
    },
    onSearch: (query, results) => {
        console.log(`Found ${results.length} results`);
    },
    onOpen: () => console.log('Search opened'),
    onClose: () => console.log('Search closed')
});

Configuration Options

Option Default Description
inputSelector .doc-search-input Search input element selector
resultsSelector .doc-search-results Results container selector
data null Custom data array (auto-indexes DOM if null)
debounceMs 150 Debounce delay in milliseconds
maxResults 20 Maximum results to display
minQueryLength 2 Minimum characters to trigger search

JavaScript API

// Access the default instance
const search = DocSearch;

// Or create a custom instance
const search = Search.create({ /* options */ });

// Methods
search.open();           // Open/focus search
search.close();          // Close results
search.clear();          // Clear input and results
search.destroy();        // Remove all event listeners

// Update search index (after DOM changes)
search.reindex();

// Search programmatically
search.search('query');

// Get current state
const isOpen = search.isOpen;

Category Icons

Customize icons for each category:

const search = Search.create({
    categoryIcons: {
        'Getting Started': 'ph-rocket',
        'Components': 'ph-cube',
        'API': 'ph-code',
        'Guides': 'ph-book-open',
        'default': 'ph-file-text'
    }
});

Icons use Phosphor Icons classes. The default key is used for categories without a specific icon.

CSS Customization

Override CSS variables to customize the appearance:

.vd-doc-search {
    /* Input styling */
    --vd-doc-search-input-bg: var(--vd-bg-secondary);
    --vd-doc-search-input-border: var(--vd-border-color);
    --vd-doc-search-input-radius: var(--vd-btn-border-radius);
    
    /* Results panel */
    --vd-doc-search-results-bg: var(--vd-bg-primary);
    --vd-doc-search-results-shadow: var(--vd-shadow-lg);
    --vd-doc-search-results-max-height: 400px;
    
    /* Result items */
    --vd-doc-search-item-hover-bg: var(--vd-bg-secondary);
    --vd-doc-search-highlight-bg: rgba(var(--vd-color-primary-rgb), 0.2);
    --vd-doc-search-highlight-color: var(--vd-color-primary);
}

Key Action

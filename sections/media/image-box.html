Image Box

A smooth "lightbox" implementation for enlarging images. Supports dual-image mode with separate thumbnails and full-size images. When users click on an image, Image Box centers it and smoothly scales it up. Dismiss by clicking, pressing ESC, or scrolling away.

Quick Start

Add the data-image-box attribute to any image you want to make enlargeable:

The component auto-initializes on page load. No JavaScript setup required.

Features

  • Dual Image Support: Use data-image-box-full-src for separate thumbnail and full-size images
  • Smooth Transitions: CSS-powered scale animation with cubic-bezier easing
  • Zoom Cursor: Magnifying glass icon on hover indicates enlargement
  • Multiple Close Methods: Click image, press ESC, or scroll away
  • Scroll Detection: Auto-closes when scrolling more than 50px
  • Backdrop Blur: Beautiful frosted glass effect behind enlarged image
  • Captions: Optional captions with data-image-box-caption
  • Accessible: Full keyboard navigation and ARIA attributes

Gallery Grid

Create responsive image galleries using the .image-box-gallery class. The grid automatically adjusts columns based on available space.

The gallery uses CSS Grid with auto-fit columns (minimum 120px). Add data-orientation="portrait" for 3:4 aspect ratio images.

JavaScript API

// Access the component
const imageBox = Vanduo.components.imageBox;

// Open an image programmatically
imageBox.open(document.querySelector('[data-image-box]'));

// Close the image box
imageBox.close();

// Re-initialize after adding new images to DOM
Vanduo.reinit('imageBox');

// Destroy and clean up
imageBox.destroy();

CSS Customization

Override CSS variables to customize the appearance:

:root {
  /* Backdrop styling */
  --image-box-backdrop-bg: rgba(0, 0, 0, 0.9);
  --image-box-backdrop-blur: 8px;
  
  /* Animation duration */
  --image-box-transition-duration: 0.3s;
  
  /* Max size of enlarged image (default: 50% of viewport) */
  --image-box-max-width: 50vw;
  --image-box-max-height: 50vh;
}

/* Gallery grid customization */
.image-box-gallery {
  gap: 21px; /* Increase spacing */
}

/* Active/selected thumbnail */
.image-box-gallery .image-box-trigger.is-active {
  outline: 3px solid var(--color-primary);
  transform: scale(1.05);
}

Data Attributes

Attribute Type Description
data-image-box Boolean Enables Image Box functionality on the image
data-image-box-full-src String URL of full-size image to display in lightbox (optional, uses src if not set)
data-image-box-src String Alternative source URL (fallback if full-src not set)
data-image-box-caption String Caption displayed below the enlarged image
data-orientation String Use "portrait" for 3:4 aspect ratio gallery thumbnails

Keyboard Shortcuts

Key Action
Enter / Space Open Image Box (when image is focused)
Escape Close Image Box
Scroll Close Image Box (when scrolled > 50px)

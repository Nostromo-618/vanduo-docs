Migration Guide

Upgrading from v1.0.x to v1.1.0 requires updating class names and initialization code. This guide will help you migrate smoothly.

Quick Migration Script

We've created a Node.js migration script to help automate the class name updates. You can find it in the migration guide:

# Open the migration guide
https://github.com/Nostromo-618/vanduo-framework/blob/main/MIGRATION.md

# Run on your project (see guide for usage)
node migrate-v1.1.0.js ./my-project

# Review changes
git diff

The script handles:

  • Adding vd- prefix to all framework classes
  • Updating CSS variable references
  • Converting initialization code

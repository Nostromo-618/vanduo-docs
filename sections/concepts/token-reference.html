CSS Token Reference

Complete reference of all CSS custom properties (variables) in Vanduo v1.1.0. Every token is overridable via :root and follows the --vd- prefix convention.

Runtime theming: All tokens below can be modified at runtime using the Theme Customizer. Open it with the button in the navbar to see changes live.

Primary Colors

--vd-color-primary: #22b8cf
--vd-color-primary-dark: #1098ad
--vd-color-primary-light: #66d9e8

Usage

.my-header {
  color: var(--vd-color-primary);
  border-bottom: 2px solid var(--vd-color-primary-dark);
}

Semantic Colors

--vd-color-success: #40c057
--vd-color-warning: #fab005
--vd-color-error: #fa5252
--vd-color-info: #228be6

Usage

.status-ok { color: var(--vd-color-success); }
.status-warn { color: var(--vd-color-warning); }
.status-fail { color: var(--vd-color-error); }

Backgrounds

--vd-bg-primary: #ffffff
--vd-bg-secondary: #f8f9fa
--vd-bg-tertiary: #e9ecef

Usage

.sidebar {
  background: var(--vd-bg-secondary);
}
.code-block {
  background: var(--vd-bg-tertiary);
}

Text Colors

--vd-text-primary: #212529
--vd-text-secondary: #495057
--vd-text-muted: #868e96

Usage

.caption {
  color: var(--vd-text-muted);
}
.heading {
  color: var(--vd-text-primary);
}

Typography

--vd-font-family-base: 'JetBrains Mono', sans-serif --vd-font-family-mono: 'JetBrains Mono', monospace --vd-font-size-base: 1rem

Usage

.custom-text {
  font-family: var(--vd-font-family-base);
  font-size: var(--vd-font-size-base);
}

Spacing (Fibonacci)

Spacing follows the Fibonacci sequence: 1, 2, 3, 5, 8, 13, 21, 34 px
--vd-spacing-1: 1px --vd-spacing-2: 2px --vd-spacing-3: 3px --vd-spacing-5: 5px --vd-spacing-8: 8px --vd-spacing-13: 13px --vd-spacing-21: 21px --vd-spacing-34: 34px

Usage

.spaced-section {
  padding: var(--vd-spacing-13);
  margin-bottom: var(--vd-spacing-21);
  gap: var(--vd-spacing-8);
}

Borders & Radius

--vd-btn-border-radius: 0.25rem --vd-card-border-radius: 0.5rem

Usage

.my-panel {
  border-radius: var(--vd-card-border-radius);
  border: 1px solid var(--vd-border-color);
}

Shadows

--vd-shadow-sm: 0 1px 2px rgba(0,0,0,0.05) --vd-shadow-md: 0 4px 6px rgba(0,0,0,0.1) --vd-shadow-lg: 0 10px 15px rgba(0,0,0,0.1)

Usage

.elevated-card {
  box-shadow: var(--vd-shadow-md);
}
.elevated-card:hover {
  box-shadow: var(--vd-shadow-lg);
}

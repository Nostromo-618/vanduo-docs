Module Formats

Vanduo v1.1.0 supports three module formats to work with any project setup, from simple static sites to modern bundlers.

IIFE

For traditional script tags. Creates a global Vanduo object. 

<script src="dist/vanduo.min.js"></script>
<script>
    Vanduo.init();
</script>

ESM

For modern bundlers and native ES modules. Enables tree-shaking.

<script type="module">
    import { Vanduo } from './dist/vanduo.esm.js';
    Vanduo.init();
</script>

CJS

For Node.js environments and older bundlers.

const { Vanduo } = require('./dist/vanduo.cjs.js');

// Initialize
Vanduo.init();

Package Exports

// package.json
{
  "exports": {
    ".": {
      "import": "./dist/vanduo.esm.js",
      "require": "./dist/vanduo.cjs.js"
    },
    "./css": "./dist/vanduo.min.css",
    "./css/components/*": "./css/components/*",
    "./js": {
      "import": "./dist/vanduo.esm.js",
      "require": "./dist/vanduo.cjs.js"
    }
  }
}

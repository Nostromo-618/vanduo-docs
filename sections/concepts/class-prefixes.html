Class Prefix Reference

Complete reference of all CSS class prefixes used in Vanduo v1.1.0.

Component Classes

Layout

  • .vd-container
  • .vd-container-fluid
  • .vd-row
  • .vd-col-*
  • .vd-col-{breakpoint}-*

Components

  • .vd-btn, .vd-btn-*
  • .vd-card, .vd-card-*
  • .vd-alert, .vd-alert-*
  • .vd-badge, .vd-badge-*
  • .vd-modal, .vd-modal-*
  • .vd-navbar, .vd-navbar-*
  • .vd-dropdown, .vd-dropdown-*

Utilities

  • .vd-text-* (colors, sizes)
  • .vd-bg-* (backgrounds)
  • .vd-m*, .vd-p* (spacing)
  • .vd-d-* (display)
  • .vd-flex-* (flexbox)
  • .vd-shadow-*
  • .vd-rounded-*
